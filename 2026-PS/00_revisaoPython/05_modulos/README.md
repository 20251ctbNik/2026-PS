A ferramenta de Transformação de Medidas é uma iniciativa de desenvolvimento em Python com o intuito de mostrar como empregar pacotes e módulos para estruturar o código. Ela executa transformações de temperatura, distância e massa, além de aplicação de componentes da biblioteca nativa e a confecção de interfaces de programação (APIs) básicas e de reaproveitamento.

A estrutura é segmentada em:
   O arquivo central, chamado main. py, mostra a aplicação dos diversos módulos.
   Dentro do diretório conversores, estão os arquivos responsáveis pelas transformações:
      de temperatura. py,
      de distancia. py e
      de massa. py
   A pasta utils tem funções de suporte para ajustes visuais (contido em formatador. py).

O sistema emprega funções para transformar unidades (como graus Celsius em Fahrenheit, quilômetros em milhas, quilogramas em libras), estabelecendo alguns valores fixos e exemplos de utilização de recursos da biblioteca padrão do Python (como math, random, datetime). Sua peopósito é instruir sobre métodos de arranjo do código, a capacidade de reutilizar funcionalidades e o modo de organizar projetos em Python.
